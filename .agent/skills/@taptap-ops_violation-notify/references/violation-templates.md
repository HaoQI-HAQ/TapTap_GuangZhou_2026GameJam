# 违规类型与处置参考

> 本文档整理自官方源文档，作为 violation-notify Skill 的核心参考。
> 禁止编造或推测任何未在此文档中出现的内容。

---

## 一、计费违规

**定义**：根据国家有关规定，未经国家新闻出版署批准的移动游戏，不得在大陆地区上线运营，即没有版号的游戏不得有任何形式的内购计费内容，包括但不限于引导充值、赞助等盈利行为。

### 1.1 偷开计费

**子类型定义**：无版号游戏私自开启付费功能或内购内容，包括但不限于引导充值、赞助等盈利行为

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、页面下架、永久不予收录 |
| 整改要求 | 删除付费相关内容后重新提审，或提交补充版号资质 |
| 备注 | 整改完成后后续不可再犯，否则有永久下架风险 |

### 1.2 多次偷开计费

**子类型定义**：已因偷开计费被处理后，再次出现同类行为，属于屡次违规

| 项目 | 内容 |
|------|------|
| 处罚措施 | 下架处理，不予恢复下载 |
| 整改要求 | 无整改机会（屡次违规） |

## 二、内容违规

**定义**：商店页素材物料或游戏内，不得含有违法内容或其他敏感信息，包括但不限于涉政、涉赌、暴力、血腥、虐待动物或儿童、淫秽、色情、性暗示及广告法中不允许使用的词汇。

### 2.1 色情内容

**子类型定义**：游戏内或商店页含有裸露身体、性行为描写、性暗示等色情或情色内容

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、删除素材、下架页面、不予收录 |
| 整改要求 | 做和谐整改后重新提审 |

### 2.2 红色血液

**子类型定义**：游戏内含有写实风格的鲜红血液、断肢、内脏等血腥暴力画面（有别于卡通或非写实风格）

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、删除素材、下架页面、不予收录 |
| 整改要求 | 排查并修改红色血液素材后重新提审 |

### 2.3 涉政内容

**子类型定义**：游戏内或商店页含有涉及政治敏感议题、国家领导人形象、历史争议事件等内容，可直接处理

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、页面下架、删除素材、不予收录 |
| 整改要求 | 删改涉政内容后重新提审；严重情况需走工单特殊审核 |
| 备注 | 涉及历任领导人、恐怖组织、疫情负面内容等需关闭下载并提交工单审核 |

### 2.4 低俗广告

**子类型定义**：游戏内嵌入含有低俗、色情擦边、欺骗性或强诱导性的第三方广告内容

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、删除素材、下架页面、不予收录 |
| 整改要求 | 替换低俗广告后重新提审 |

## 三、知识产权

**定义**：商店页素材物料或游戏内容（含角色立绘、场景设计、UI 界面、游戏名称等），不得侵犯他人知识产权，包括但不限于直接抄袭知名 IP 的角色形象或美术内容、刻意模仿热门 IP 误导用户，以及使用 IP 内容但无法提供合法授权证明。

### 3.1 素材侵权抄袭

**子类型定义**：直接复制或高度相似地使用他人享有著作权的美术素材、角色形象、UI 设计等，构成著作权侵权

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、删除素材、下架页面、不予收录 |
| 整改要求 | 确认并上传授权资质，或撤换侵权素材 |

### 3.2 擦边碰瓷热门 IP

**子类型定义**：游戏名称、图标、角色等刻意模仿知名 IP 的视觉元素或命名方式，借助其知名度吸引用户，但未直接抄袭

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、删除素材、下架页面、不予收录 |
| 整改要求 | 确认并上传授权资质，或撤换侵权素材 |

### 3.3 游戏 IP 授权存疑

**子类型定义**：游戏使用了某知名 IP 的相关内容，但无法提供真实有效的版权方授权证明文件

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、删除素材、下架页面、不予收录 |
| 整改要求 | 确认授权资质，或撤换侵权素材；可走工单咨询 |

## 四、分发违规

**定义**：游戏在 TapTap 平台内，不得通过任何方式引导用户绕过平台渠道获取安装包，包括但不限于提供外部下载链接或二维码、引导用户通过社交群组获取安装包等行为。

### 4.1 引导外部包下载

**子类型定义**：在游戏内弹窗、商店页描述或开发者公告中，提供非 TapTap 平台的外部下载链接、二维码或文字引导

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、删除素材、下架页面、不予收录 |
| 整改要求 | 去除平台包的外链内容后重新提审 |

### 4.2 引导 Q 群下载

**子类型定义**：引导用户加入 QQ 群、微信群等社交群组，并在群内提供游戏安装包的下载渠道

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、删除素材、下架页面、不予收录 |
| 整改要求 | 修改/删除引导外部包下载的相关内容，建议在平台进行开测 |

## 五、运营违规

**定义**：开发者在 TapTap 平台的运营行为须遵守平台规则，不得扰乱平台正常秩序，包括但不限于诱导好评、频繁改名、滥用测试计划、发布雷同游戏等行为。

### 5.1 引导好评

**子类型定义**：通过奖励礼包码、道具、积分、现金等利益诱导，或通过强制弹窗等方式，引导用户给予正面评价

| 项目 | 内容 |
|------|------|
| 处罚措施 | 删除违规评价、关闭下载 |
| 整改要求 | 停止站内外引导好评行为 |
| 备注 | 此类行为可能导致站内玩家抵制，给游戏带来负面口碑影响 |

**关键判定要点**：强制弹窗本身并不违规，但利用强制弹窗进行好评诱导才属于违规。判断标准在于弹窗的目的和内容是否构成好评引导，而非弹窗形式本身。

### 5.2 多次更换游戏名

**子类型定义**：短时间内频繁修改游戏名称，可能用于混淆审核、规避处罚记录或蹭热点关键词

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、页面下架、删除素材、不予收录 |
| 整改要求 | 停止反复更名行为 |

### 5.3 高频使用封闭式测试计划

**子类型定义**：将封闭式测试作为长期运营手段，频繁开启或续期测试计划

| 项目 | 内容 |
|------|------|
| 处罚措施 | 排行榜屏蔽、暂停封闭式测试能力、限制曝光、限制上架 |
| 整改要求 | 可改用开放式测试；建议同时保持预约；违规限制可在 3 个月后申诉解除 |
| 备注 | 持续迭代高质量内容可申请撤销限流措施；可联系对接运营或提交工单申诉 |

### 5.4 游戏雷同重复


**别名/俗称**：重复入库（业务中常用此称呼）
**子类型定义**：同一开发者发布多款玩法、美术或核心内容高度雷同的游戏，或与平台已有游戏高度相似

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、页面下架、永久不予收录 |
| 整改要求 | 勿上传多个内容相似、资质相同的游戏 |
| 备注 | 跨厂商账号上传雷同游戏可能导致账号注销、游戏名永久拒绝收录、限制注册等 |

## 六、资质问题

**定义**：在 TapTap 平台发布游戏的开发者须具备合法有效的经营资质，游戏须满足国家监管要求，包括但不限于资质材料真实有效、具有合法分发主体、持有有效版号、防沉迷机制合规。

### 6.1 资质异常

**子类型定义**：开发者提交的营业执照、软件著作权、ICP 备案等资质文件存在伪造、过期或与实际信息不符的情况

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、删除资质、下架页面、不予收录 |
| 整改要求 | 确保软著、版号资质与游戏标题保持统一，在开发者中心-游戏资质中修改提审 |

### 6.2 没有分发主体

**子类型定义**：开发者账号无对应的合法注册企业主体，或分发主体信息缺失、无效

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、页面下架、删除素材、不予收录 |
| 整改要求 | 完善厂商注册信息成为认证开发者后，再重新上架游戏 |

### 6.3 境外无版号

**子类型定义**：境外开发商的游戏在大陆上线运营，但未取得国家新闻出版署核发的游戏版号

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、页面下架、删除素材、不予收录 |
| 整改要求 | 建议将入驻信息资料修改为大陆居民身份证 |

### 6.4 版号运营单位已吊销

**子类型定义**：游戏持有的版号所对应的运营主体企业已被吊销营业执照或依法注销

| 项目 | 内容 |
|------|------|
| 处罚措施 | 下架处理 |
| 整改要求 | 联系版号运营单位核实情况，上传提供有效的授权书 |

### 6.5 防沉迷不合规

**子类型定义**：游戏未按国家规定接入实名注册认证系统，或未落实未成年人游戏时长、消费限额等防沉迷措施

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、页面下架、删除素材、不予收录 |
| 整改要求 | 修复防沉迷系统，确保不可跳过 |

## 七、游戏质量

**定义**：在 TapTap 平台发布的游戏须达到基本可玩标准，不得损害用户体验或扰乱平台内容生态，包括但不限于游戏可正常运行、敏感词过滤有效、不批量上架低质游戏、不发布盗版游戏。

### 7.1 游戏不能正常游玩

**子类型定义**：游戏存在频繁崩溃、卡死、无法加载、核心功能无法使用等严重影响正常体验的问题

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、删除素材、下架页面、不予收录 |
| 整改要求 | 修复游戏问题后在开发者中心重新提审 |
| 备注 | 优先发通知+关闭下载；量级较大的游戏可有 1 个工作日缓冲时间 |

### 7.2 未屏蔽敏感词

**子类型定义**：游戏内聊天、昵称等用户生成内容场景，未部署有效的敏感词过滤机制，或过滤系统形同虚设

| 项目 | 内容 |
|------|------|
| 处罚措施 | 关闭下载、页面下架、删除素材、不予收录 |
| 整改要求 | 完善敏感词屏蔽系统 |

### 7.3 大量低质游戏

**子类型定义**：同一开发者账号批量发布使用相同模板或引擎简单套壳、缺乏独立游戏内容的低质量游戏

| 项目 | 内容 |
|------|------|
| 处罚措施 | 参见「低质厂商处罚」（8.2） |

### 7.4 盗版抄袭

**子类型定义**：游戏对市场已有游戏的核心玩法、美术资产、代码等进行整体或大量复制，侵犯原作版权

| 项目 | 内容 |
|------|------|
| 处罚措施 | 参见「盗版抄袭厂商处罚」（8.1） |

## 八、账号/厂商级

**定义**：开发者账号须由合法主体持有并承担相应责任，不得存在批量违规行为，包括但不限于多次发布盗版内容、批量上架低质游戏等。

### 8.1 盗版抄袭厂商处罚

**子类型定义**：多次发布盗版、抄袭游戏，累计违规达到平台处罚阈值

| 项目 | 内容 |
|------|------|
| 处罚措施 | 违规游戏：关闭下载、页面下架、限流分发、不予收录；开发者账号：限流、下架乃至注销 |
| 整改要求 | 联系对接运营或提交工单申诉 |
| 备注 | 违规事项包括但不限于内容版权违规、广告体验极差、品质低劣等引发大量玩家投诉 |

### 8.2 低质厂商处罚

**子类型定义**：批量上架大量低质游戏，严重消耗平台审核资源、损害用户体验

| 项目 | 内容 |
|------|------|
| 处罚措施 | 违规游戏：关闭下载、页面下架、限流分发、不予收录；开发者账号：限流、下架乃至注销 |
| 整改要求 | 联系对接运营或提交工单申诉 |
| 备注 | 违规事项包括但不限于内容版权违规、广告体验极差、批量上传低质游戏等引发玩家投诉 |

## 十、流量受限标签（非违规）

**说明**：流量受限不是违规处罚，而是平台对游戏展示分发的限制状态。所有通过 Maker 上架的游戏**默认不分发**（即默认流量受限）。

**解除流量受限的方式**：

| 序号 | 条件 | 说明 |
|------|------|------|
| 1 | 完善商店页素材 | 确保以下素材已上传：简介、开发者的话、游戏截图、宣传图、**实机视频**（尤其重要）、首页推荐素材 |
| 2 | 论坛发帖申请评审 | 游戏上架后，前往论坛**「游戏上新」**板块发帖，帖子需包含游戏链接及游戏分享内容，等待编辑同学评审 |
| 3 | 风控原因整改 | 如果是因风控原因受限，需根据风控提供的具体证据和理由进行对应修改 |

**注意**：
- 条件 1 和条件 2 是所有 Maker 游戏解除流量受限的标准流程
- 条件 3 仅适用于因风控原因被额外限制的情况

**解除后效果**：恢复首页推荐流量 + 新品榜入榜资格 + 3 天冷启动流量

## 附录：通用处罚措施说明

以下处罚措施均引用自官方通知中的原文表述：

| 处罚措施 | 说明 |
|---------|------|
| 关闭下载 | 游戏无法被下载，但页面仍可见 |
| 页面下架 | 游戏页面不可见 |
| 删除素材 | 移除违规的商店页素材 |
| 永久不予收录 | 游戏永久无法在平台上架 |
| 不予恢复下载 | 多次违规后不再允许恢复 |
| 限制曝光/限流 | 减少游戏在平台的展示机会 |
| 限制上架 | 开发者无法发布新游戏 |
| 账号注销 | 开发者账号被注销 |
| 排行榜屏蔽 | 游戏从排行榜中移除 |
| 暂停测试 | 无法使用封闭式测试能力 |
